# GraphicsV1 - 3D Wireframe renderering from scratch

## Camera Controls
- WASD for Forward/Left/Back/Right
- F for up, C for down
- Arrow keys for look direction

## Limitations
- For each 3D shape, it will connect every vertex to every other vertex
- Only renders edges
